# PostureLock AI

## Inspiration

We wanted to help students and developers stay focused and maintain better posture while spending long hours on their laptops. Many people slowly start slouching or losing focus without noticing, so we created a tool that can detect it and provide a real-time reminder.

## What It Does

PostureLock is a background laptop application that uses your webcam and AI to detect when you are slouching or leaving your focused state. It learns your proper posture, waits before reacting to avoid false alerts, and reminds you to sit up and get back on track.

## How We Built It

We built PostureLock using computer vision and AI models to track face and body movements through the webcam. The app uses real-time posture detection, a calibration system, and a smart alert system that runs locally on the user's computer.

## Challenges We Ran Into

One of the biggest challenges was making the AI accurately detect posture without constantly sending incorrect alerts. We also had to improve performance, handle different lighting conditions, and make sure the application could run smoothly in the background.

## Accomplishments That We're Proud Of

We created a working AI application that can detect posture changes live and respond in real time. We are proud that PostureLock is not just a concept, but a functional tool that helps users improve focus and posture.

## What We Learned

We learned how to build real-time AI systems, process live camera data, and design technology that adapts to different users. We also learned how small design decisions can make an application more useful and user-friendly.

## What's Next for PostureLock

We plan to improve detection accuracy, add more focus features, and make PostureLock a smarter personal productivity coach that helps users stay focused throughout their work sessions.
